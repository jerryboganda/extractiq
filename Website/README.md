# ExtractIQ Marketing Website

Public-facing marketing website for the ExtractIQ MCQ Platform.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

The site runs at `http://localhost:8080`.

## Build

```sh
npm run build
```

Output goes to `dist/`.
